# Voice Guide — Anti-AI Rules, Banned Words, Style

> The last gate before anything is shown. Every draft gets checked against this file.
> Drafted from `CLAUDE.md` rule 7 ("Direct + specific. Expert advice, real numbers, no
> fluff") and the FORGE honesty doctrine. **[REFINE]** to taste.

---

## The one-line voice

**Direct, specific, honest builder.** Talks like someone who actually shipped the thing —
real numbers, real screenshots, no fluff, no hype. Confident, not salesy.

- **Register:** casual-professional. Contractions yes. Slang sparingly and only if real.
- **Person:** first person ("I built…", "here's what happened"). Talk TO one viewer.
- **Sentence length:** short. Punchy. Vary it. One idea per sentence.
- **Proof:** show, don't tell. "Here's the dashboard" beats "it's powerful."

---

## Anti-AI-slop rules (hard)

1. **No em-dash-and-"it's not X, it's Y" cadence** as a crutch. Use it once, maybe.
2. **No hedging.** Kill "arguably," "essentially," "in many ways," "when it comes to."
3. **No throat-clearing intros.** Never open with "In today's fast-paced world" or
   "Let's dive in." Start on the hook.
4. **No listicle padding.** If a point isn't real, cut it. 3 real > 7 filler.
5. **Concrete nouns + real numbers.** "8 agents across 3 businesses" not "a suite of
   powerful tools."
6. **One angle per asset.** A thumbnail/hook makes ONE promise, not three.
7. **Never invent proof.** No fabricated metrics, testimonials, or "results." **[REFINE]**
   tag anything unverified — it does not ship as fact.

> Note: "leverage" is fine as a real concept ("compounding leverage"), banned as fluff.

## Formatting defaults

- **Titles/thumbnails:** ≤ 5 words on the image. ALL CAPS or Title Case, one accent word.
- **Hooks:** first line must work with zero context. No setup.
- **CTAs:** one, specific, action-first. [REFINE — the real CTA from `business_overview.md`.]
- **Emoji:** sparingly, only where it earns attention. Never decorative rows.

## Voice check (run before showing any draft)

- [ ] Does it make ONE promise?
- [ ] Is every claim real or **[REFINE]**-tagged? (No invented proof.)
- [ ] Any banned words? Cut them.
- [ ] Would a skeptical builder believe it? If it smells like hype, rewrite.
- [ ] Does it sound like me, or like default AI? If default, rewrite harder.
